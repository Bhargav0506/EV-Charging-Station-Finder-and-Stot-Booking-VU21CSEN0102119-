# EV Charging Station Finder and Slot Booking

## Overview
This project aims to develop a comprehensive system for electric vehicle (EV) owners to easily locate and book charging stations. The application provides a user-friendly interface to search for available charging stations based on location and book slots in advance.

## Features
- **Charging Station Finder**: Users can search for charging stations by city or location.
- **Slot Booking**: Users can book available slots at selected charging stations.
- **Multiple Filters**: Charging stations can be filtered based on various criteria such as charging type, availability, and ratings.
- **Advance Booking**: Users can book charging slots in advance to ensure availability.
- **User Management**: Users can create accounts and manage their charging history and preferences.

## Technologies Used
- **Flutter**: A popular open-source mobile app development framework used for building cross-platform applications.
- **Firebase**: A comprehensive app development platform that provides a suite of tools and services for building high-quality apps.
- **Google Maps API**: Used for integrating maps and location-based features into the application.


Set up Flutter development environment: Follow the official Flutter installation guide for your operating system.
Configure Firebase: Create a new Firebase project and enable the necessary services (Authentication, Firestore, etc.). Download the google-services.json file and add it to your project's android/app directory.
Install dependencies:
bash
flutter pub get

Run the app:
bash
flutter run

Usage
Sign up or log in to the application using your email and password.
Search for charging stations by entering your location or selecting from a list of cities.
Filter the results based on your preferences (e.g., charging type, availability).
Select a charging station and book a slot by choosing an available time slot.
Manage your bookings and view your charging history from the user dashboard.
